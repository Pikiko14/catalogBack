
# Crew Web Catalog

Catalogo digital inteligente para pequeñas y medianas empresas.




## Deployment

Para instalar el proyecto en tu entorno local

```bash
  sudo Git clone https://github.com/Pikiko14/catalogBack
  cd catalogBack
  sudo npm install
  sudo mkdir uploads
  sudo mkdir catalogues
  sudo mkdir categories
  sudo mkdir images
  sudo mkdir pages
  sudo mkdir pdfs
  sudo mkdir products
  sudo mkdir profile
  npm run dev
```

Para instalar el proyecto en tu entorno producción

```bash
  sudo Git clone https://github.com/Pikiko14/catalogBack
  cd catalogBack
  sudo npm install
  sudo mkdir uploads
  sudo mkdir catalogues
  sudo mkdir categories
  sudo mkdir images
  sudo mkdir pages
  sudo mkdir pdfs
  sudo mkdir products
  sudo mkdir profile
  npm run build
```
